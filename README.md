# シューティングこうかとん

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* 主人公キャラクター（こうかとん）をマウスで操作し、敵UFOを打つシューティングゲーム

## ゲームの遊び方
* 矢印キー ↑ ↓ ← → でこうかとんを操作する
* Spaceキー でビーム弾を消費しビームを放つ
* 0キーを 2秒長押し → 離す とチャージビームを放つ
* Eキー で電磁パルスを放つ（スコア30消費）
* Enterキー で重力場を放つ（スコア100消費）
* 敵を倒すとスコアを10、ビーム弾を5獲得
* 敵を5体倒すとボスが出現

## ゲームの実装
* 敵が降ってくるときの左右移動：C0A25052
* 回復手段:C0A25020 体力が減ったらマックスになるまで左からランダムに回復アイテムが流れてくる
* ボスの実装：C0C25018
* 弾数の実装（初期弾数 5 発、敵を倒すたびに 5 発装填）：C0A25116
* チャージビーム（ためてビームを 5 発発射する）:C0A25112
